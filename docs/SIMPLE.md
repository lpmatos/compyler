# Linguagem de Programação SIMPLE

A linguagem de programação SIMPLE, como o próprio nome indica, é uma linguagem simples, mas ainda sim poderosa e de alto nível, semelhante às versões iniciais da conhecida linguagem BASIC:

- Cada instrução da linguagem SIMPLE consiste em um número de linha e um comando da linguagem. 
- Os números de linha devem aparecer em ordem crescente. 
- Cada comando inicia com uma das seguintes palavras reservadas: `rem`, `input`, `let`, `print`, `goto`, `if/goto` ou `end`.

- A atividade pode ser realizada em dupla:
  - Nesse caso, a atividade deve ser submetida por um aluno, e na documentação apresentar o nome/matrícula dos integrantes da dupla;
- A atividade pode ser implementada na linguagem de programação de preferência do aluno:
  -  documentação deve constar as instruções para compilação e execução do programa submetido;
- As atividades que apresentarem mais do que 15% de plágio serão desconsideradas, em comparação a atividades externas ou internas;
- As atividades que não compilarem e/ou executarem serão desconsideradas;
- A atividade será executada no sistema operacional Ubuntu 20.04.2 LTS;
  - reconhecimento do erro num programa SIMPLE contendo um erro (vale 0,5 ponto);
  - reconhecimento dos erros num programa SIMPLE contendo dois erros (vale 0,5 ponto);
  - reconhecimento dos erros num programa SIMPLE contendo três erros (vale 0,5 ponto);
  - reconhecimento dos erros num programa SIMPLE contendo quatro erros (vale 0,5 ponto);
  - os erros serão sintáticos e/ou semânticos.
